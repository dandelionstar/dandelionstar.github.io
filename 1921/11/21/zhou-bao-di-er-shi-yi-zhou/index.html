生命的律动:

4、总结：

4.1 评述：

  • 运动量充足
    • 两次健身房
    • 打球三次
    • 一次体测
  • 作息规律稳定
    • 10点嗑褪黑素
  • 12点半入睡
    • 8点多醒来并起床
  • 学习了一些新东西
    • 股票
    • vue
    • 练车
      • 倒车入库
      • 侧方
  • 情绪基本稳定
    • 无敏感脆弱情绪
    • 有一点轻燥
  • 新设备
    • 鸿蒙matepad11

4.2 评分

项目 分数
健康指数 6
kpi 0
总分 1

5、反思：

5.1 问题

  1. 没有准备四级考试
    • 没看刘晓燕网课
    • 没记单词
    • 没做四级题
  2. 健身不过用心
    • 健身动作走神
    • 没有自己的安排和计划
  3. 写博客的意愿与能力问题
    • 没意愿写
    • 写作太慢
  4. 效率问题
    • 容易走神，注意力不集中
    • 对效率机制的构建投入的注意力不充足
  5. 迭代主题没有实现
    • 细化评分机制两周都没有落实
  6. 做的事情kpi有很大出入
    • 没准备四级
    • 做了很多主线相关的事

5.2 改进

  1. 坚持周三，周五的审计；
    • 审计可以更好落实kpi
    • 审计可以灵活修改kpi
  2. 跟踪kpi和任务清单
    • 正视自己的效率和能力
    • 发现哪些任务无法按时完成
    • 发现效率问题并改进

6、下周KPI：

6.1 健康主题[50%]

  1. 至少去4次健身房
    • 周一
    • 周三
    • 周五
    • 周天
  2. 打球3次
    • 周二
    • 周四
    • 周六
  3. 入睡超过1点的次数不多于两天；最晚不能超过2点
  4. 起床晚于8:30点的次数不超过两天；最晚不能超过10点
  5. 每天按时吃饭；每周不吃早点的天数不超过三天；

6.2 学习主题[15%]

  1. vue

    • 复习前面大纲

    • toDoList案例

  2. 四级

    • 背单词[怎么个背法?]
    • 看刘晓燕四级网课[多少?]

  3. 理财

    • 敖丙的视频每天刷一个

6.3 活动主题[15%]

  1. 超过4天练习冥想
  2. 练车！练车！练车！
    • 3小时 x 7
  3. 痛定思痛
    • 迭代构建情绪监控机制和情绪调节
    • 时间管理机制
    • 如何改善自己的生活方式
  4. dandelion需求文档开发
    • 佛系开发
    • 归档

6.4 迭代主题[20%]

  1. 细化评分机制
    • 按比例计算得分
    • 每一项打出分数
  2. 效率机制
    • 限时
